Project Radiance

Project Radiance aimed to address the lack of public education on radiation by creating a publicly accessible relational database hosted on AWS. This database compiled and organized global radiation measurements from various monitoring locations worldwide. The project utilized SQL for querying and managing the database, Python and R for Extraction, Transformation, and Loading (ETL) of data, and employed Python, R, and Power BI for creating intuitive data visualizations and projections.
